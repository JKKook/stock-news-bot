## 0. 한 줄 결론

**가장 가치 있는 업그레이드는 "헤드라인을 더 많이"가 아니라, ①최신순이 아닌 '이례성(surprise)·임팩트' 우선순위, ②"무엇이 바뀌었고 왜 중요한가"의 so-what 레이어, ③푸시 알림이 증폭하는 행동재무 함정에 대한 설계적 방어 — 이 세 가지다.**

근거(검증됨):
- 시장은 뉴스의 *예상 대비 편차(surprise)* 에 반응하고, **투자자 관심이 이미 높을 때 같은 크기의 서프라이즈가 더 크게 움직인다** (Fed FEDS 2025-022; Cleveland Fed: 2022–24 CPI 서프라이즈당 금리 민감도 ~4배). → 순위 신호 = 서프라이즈 크기 + 현재 관심도.
- **선정적/극단 헤드라인일수록 이미 (과)반영되고 이후 되돌림** (Kwon & Tang, *Review of Economic Studies*). → "규모 + 이미 반영 여부 + 되돌림 위험" 주석 필요, 특히 **보유 종목**(Hartzmark et al., QJE: 사람은 *가진 종목* 뉴스에 과잉반응).
- **뉴스에 떴다는 이유만의 매수는 초과수익이 없거나 음(−)** 이고, 앱식 단순 알림은 **군집행동을 집중**시킨다 (Barber & Odean RFS 2008; Robinhood 연구 JF 2022: 극단 군집 매수는 5일 −6% 비정상수익). → 임팩트 게이팅·묶음 다이제스트·"이미 반영/되돌림 위험" 맥락이 **1급 기능**.

## 1. 우선순위 로드맵 (임팩트 × 노력)

### 🟢 P0 — 즉시·무료·고임팩트 (LLM/유료 API 불필요)

**P0-1. 관심종목 price action 연결** ⭐ 최우선
- 뉴스 옆에 해당 종목의 **당일 등락% · 거래량(평소 대비) · 52주 위치**를 붙인다.
- *왜*: 헤드라인의 so-what 공백을 즉시 메움. "삼성전자 실적 주목" → "삼성전자 +3.2%, 거래량 2.1×, 52주 고점 −8%"면 판단 재료가 된다. 뉴스↔가격 연결이 현재 최대 공백.
- *데이터*: **yfinance**(이미 야후 chart API 사용 중 — 확장만). 무료.
- *근거*: 서프라이즈·관심 증폭(FEDS 2025-022), 보유종목 과잉반응(QJE).

**P0-2. 속보 순위: 최신순 → 이례성(surprise) 게이팅**
- 뉴스 속보를 시간순이 아니라 **가격·거래량 급변을 동반하는지**로 우선순위/필터. (지수 급변은 이미 밴드로 처리 중 → 뉴스에도 같은 원리 확장.)
- *왜*: 최신성 편향이 노이즈·군집행동을 키움. 임팩트 게이팅이 방어.
- *데이터*: yfinance 거래량/가격. 무료.
- *근거*: Barber & Odean, Robinhood JF 2022.

**P0-3. 행동재무 가드레일 (설계로 방어)**
- 각 속보/급변에 **"⚠️ 이미 반영/되돌림 가능성" 맥락 라벨** + **"매매신호 아님" 면책** 상시 표기.
- 개별 핑 대신 **묶음 다이제스트**(속보를 짧은 창으로 배칭). 이미 만든 사건지문 dedup과 결합.
- *왜*: 과알림 → 알림 피로·군집행동. 극단 헤드라인 되돌림 경고는 사용자를 오버트레이딩에서 보호.
- *근거*: Kwon & Tang(되돌림), Barber&Odean(군집), 알림피로 문헌.

**P0-4. 경제·실적 촉매 캘린더 ("예정 촉매" 섹션)**
- FOMC·CPI·고용·주요 어닝 등 **다가오는 이벤트**를 임팩트 3단계 별점(★~★★★)으로.
- *왜*: 투자 판단의 실제 앵커는 "예정 촉매". 단기·장기 모두에 유효(계층화의 중기 축).
- *데이터*: Finnhub 무료(경제/어닝 캘린더, 60 req/min), TradingEconomics/Investing.com(별점 모델), 어닝은 FMP/Finnhub 무료 티어.
- *근거*: Investing.com 3단계 임팩트 별점(검증된 이식 가능 템플릿).

### 🟡 P1 — LLM "so what" 레이어 (Claude API, 소액 비용)

**P1-5. WIIM식 3줄 요약 — "무엇이 바뀌었나 / 왜 중요한가 / 무엇을 지켜볼까"**
- 같은 사건 헤드라인 클러스터 → LLM이 3줄로 압축. (Benzinga "Why Is It Moving"의 직접 이식 템플릿.)
- ⚠️ **전제: 품질 필터 먼저.** 불필터 뉴스를 그대로 요약하면 **무(無)뉴스 기준선보다 나빠진다**(NeurIPS 2024: 필터 시 오차 −17%). 관련성·임팩트로 거른 뒤 요약.
- ⚠️ **환각 방어**: 원문 grounding, 확신도 게이팅, 사실 단정 금지("~로 보임"), 면책.
- *모델 권장*: **Claude Haiku 4.5**(빠르고 저렴 — 뉴스 요약에 적합).
- *근거*: LLM 구조화 유효(arXiv 2407.15788, NeurIPS 2024) but not turnkey(LSEG).

**P1-6. 사건 → 종목/섹터 영향 태깅**
- LLM로 사건에 영향받는 티커·방향·기간(단기/장기) 태그.
- ⚠️ **주의**: per-ticker *감성* 태깅은 반증됨 — **차별화 요소가 아니라 table-stakes.** 여기에 과투자 금지. 방향/기간/영향종목 매핑에 집중.
- *근거*: 반증된 주장 2건(감성 태깅 novelty, FinancialBERT 우위) — 로드맵에서 제외.

### 🔵 P2 — 심리·수급 보강 & 계층화

**P2-7. 국내 수급 지표 (CNN 공포탐욕의 미국 편향 보완)**
- **투자자별 순매수(외국인·기관·개인)**, 신용잔고, 프로그램매매를 국내 심리·수급 지표로.
- *데이터*: **pykrx**(KRX 무료), KRX Open API.
- *근거*: 현재 심리 지표가 US only라는 공백(오픈 퀘스천). ※ 단, 행동재무 근거는 대부분 미국 시장 기반 → 국내 적용은 검증하며 도입.

**P2-8. 호라이즌 계층화**
- 섹션을 **단기(속보·price action) / 중기(촉매 캘린더) / 장기(펀더멘털·테마)** 로 분리 → 혼합 사용자가 자기 층만 보게.

## 3. 반드시 지킬 원칙 (검증에서 도출)

1. **볼륨 최대화 금지 → 관련성·임팩트 필터가 먼저.** 불필터 뉴스는 판단을 *악화*시킨다.
2. **최신순 ≠ 중요순.** surprise·거래량 동반이 우선순위 신호.
3. **극단 헤드라인엔 되돌림 경고.** 특히 보유 종목.
4. **알림은 묶고 게이팅.** 개별 핑은 군집행동을 부른다.
5. **LLM은 turnkey 아님.** grounding·확신도 게이팅·면책 필수.
6. **감성 태깅은 차별점 아님(table-stakes).** 여기 과투자 금지.

---

## 4. 남은 검증 과제 (도입 전 확인)

- 무료 소스의 **실제 레이트리밋 × GitHub Actions 크론** 궁합(특히 yfinance/pykrx 안정성).
- 미국 기반 행동재무 결과의 **국내 시장 재현성** — pykrx 지표로 국내 검증.
- 임팩트 게이팅 **임계값**과 다이제스트 **배칭 주기** 튜닝(단기 vs 장기 혼합 대응).
- LLM 출력 **환각 방어**의 프로덕션 설계(grounding/human-in-loop).

---

## 5. 권장 착수 순서

**P0-1(가격 연결) → P0-3(가드레일) → P0-4(촉매 캘린더) → P0-2(임팩트 게이팅)** 를 먼저.
전부 **무료 데이터 + 기존 스택**으로 가능하고, 근거가 가장 강하며, LLM 비용/환각 리스크가 없다.
그 다음 효과를 보고 **P1(LLM so-what 레이어)** 로 진입.
