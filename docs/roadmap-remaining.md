## 🥇 가치 높음 (정확도·신속성 직결)

### R2. 속보 뉴스 자체의 거래량 게이팅
- **무엇**: 지금은 속보가 관심종목을 *지목할 때만* 가격 확증. 이를 확장해 **모든 속보 후보를 관련 종목 거래량과 매핑**해 '가격·거래량 미반응' 뉴스는 후순위/억제.
- **왜**: 자극적이지만 시장이 무시하는 뉴스의 노이즈를 더 줄임.
- **난이도**: 상 (뉴스 제목 → 티커 매핑의 일반화가 어려움 — 지금은 관심종목 이름 매칭만).

### R3. 국내 경제 캘린더 — 실제 발표일
- **무엇**: 현재 한국 지표(금통위·CPI)는 **큐레이션 추정일**(`config.KR_ECONOMIC_EVENTS`). BOK·통계청 공식 발표일로 교체.
- **왜**: 촉매 캘린더의 정확도. **막힘**: 무료 '예정 발표일' API 부재 → BOK/통계청 공식 일정 페이지 스크래핑 또는 연 1회 수기 갱신.
- **난이도**: 중. **리스크**: 스크래핑 취약성.

## 🥈 분석 보강

### R4. 의미 dedup을 섹터·관심종목 뉴스까지 확대
- 현재 임베딩 dedup은 **헤드라인만** 적용. 테마별 소식·관심종목 뉴스에도 확대하면 중복↓.
- 난이도: 하~중 (기존 `semantic.keep_indices` 재사용). 비용: 임베딩 호출 소폭 증가.

### R5. 국내 '개인' 순매매 + 시장 전체 수급
- 현재 국내 수급은 **외국인·기관**만(네이버 개별종목). 개인 순매매는 표에 없어 제외됨.
- 시장 전체(코스피/코스닥) 투자자별 순매수를 더하면 **국내 심리 지표**(CNN 공포탐욕의 미국 편향 보완)로 승격.
- 난이도: 중 (네이버 시장 매매동향 페이지 파싱). 리스크: 스크래핑.

### R6. 코스피200 야간선물
- 현재 야간선물은 **나스닥선물(NQ=F)** 만. 한국 코스피200 야간선물은 무료 야후 심볼이 마땅치 않아 보류.
- 대안: 증권사/거래소 데이터 확보 시 추가.
